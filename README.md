Start the app

```bash
npm run dev

```

Initialize Prisma

````bash
npx prisma init


Remove everything in the db

```bash
npx prisma migrate reset

````

Add new primsa fields in schema to node modules

```bash
npx prisma generate

```

Push fields to the db

```bash
npx prisma db push

```

Run Prisma Studio to view the current prisma database

```bash
npx prisma studio

```
